# List All Models

Returns every model with its identity info, Elo/winRate/speed rankings nested by arena and category, and tournament ranking distribution (1st–4th place finishes). One call to get everything — no need to iterate over individual leaderboard endpoints.

<Info>
This is the recommended endpoint if you need a complete snapshot of all Design Arena rankings. Responses are cached for **5 minutes**. Ranking distribution data is pre-computed and refreshed daily.
</Info>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of model objects

  <Expandable title="Model properties">
    <ResponseField name="id" type="string">
      Internal model identifier (stable across updates)
    </ResponseField>
    <ResponseField name="displayName" type="string">
      Human-readable model name
    </ResponseField>
    <ResponseField name="provider" type="string">
      Model provider (e.g., "anthropic", "openai", "google")
    </ResponseField>
    <ResponseField name="openRouterId" type="string | null">
      OpenRouter model ID for cross-referencing. `null` if the model is not available on OpenRouter (proprietary platforms, image/video generators, deprecated models).
    </ResponseField>
    <ResponseField name="rankings" type="object">
      Nested rankings keyed by arena, then category. Only arena/category combinations where the model has sufficient data will appear.

      <Expandable title="Rankings structure">
        Each key is an arena ID (`models`, `builders`, `agents`), containing an object where each key is a category ID:

        <ResponseField name="elo" type="integer">
          Elo rating (higher is better, base 1200)
        </ResponseField>
        <ResponseField name="winRate" type="number">
          Win rate percentage (0–100)
        </ResponseField>
        <ResponseField name="avgGenerationTimeMs" type="integer | null">
          Average generation time in milliseconds
        </ResponseField>
      </Expandable>
    </ResponseField>
    <ResponseField name="rankingDistribution" type="object">
      Tournament placement counts across all completed tournaments the model has participated in. Excludes removed/flagged tournaments.

      <Expandable title="Ranking distribution properties">
        <ResponseField name="first" type="integer">
          Number of times the model finished 1st place
        </ResponseField>
        <ResponseField name="second" type="integer">
          Number of times the model finished 2nd place
        </ResponseField>
        <ResponseField name="third" type="integer">
          Number of times the model finished 3rd place
        </ResponseField>
        <ResponseField name="fourth" type="integer">
          Number of times the model finished 4th place
        </ResponseField>
        <ResponseField name="total" type="integer">
          Total tournaments the model has participated in (`first + second + third + fourth`)
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  <Expandable title="Meta properties">
    <ResponseField name="lastUpdated" type="string | null">
      ISO timestamp of the most recent leaderboard calculation across all arenas
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO timestamp of when this response was generated
</ResponseField>

## Examples

<RequestExample>
```bash curl
curl -H "Authorization: Bearer sk_live_your_api_key_here" \
  https://www.designarena.ai/api/v1/models
```

```javascript JavaScript
const response = await fetch(
  'https://www.designarena.ai/api/v1/models',
  { headers: { 'Authorization': 'Bearer sk_live_your_api_key_here' } }
);
const { data } = await response.json();

// Find a specific model
const claude = data.find(m => m.id.includes('claude'));
console.log(claude.rankings.models.website.elo);
console.log(claude.rankingDistribution.first); // 1st place finishes
```

```python Python
import requests

response = requests.get(
  'https://www.designarena.ai/api/v1/models',
  headers={'Authorization': 'Bearer sk_live_your_api_key_here'}
)
models = response.json()['data']

# Sort by website Elo
website_models = [
    m for m in models
    if 'models' in m['rankings'] and 'website' in m['rankings']['models']
]
website_models.sort(key=lambda m: m['rankings']['models']['website']['elo'], reverse=True)
for m in website_models[:10]:
    dist = m['rankingDistribution']
    print(f"{m['displayName']}: Elo {m['rankings']['models']['website']['elo']}, "
          f"1st: {dist['first']}, 2nd: {dist['second']}")
```

## Usage Patterns

### Cross-reference with OpenRouter pricing

```python
import requests

headers = {'Authorization': 'Bearer sk_live_your_api_key_here'}

# Get Design Arena rankings
da_models = requests.get(
    'https://www.designarena.ai/api/v1/models', headers=headers
).json()['data']

# Get OpenRouter pricing
or_models = requests.get('https://openrouter.ai/api/v1/models').json()['data']
or_pricing = {m['id']: m['pricing'] for m in or_models}

# Combine quality + cost
for m in da_models:
    if m['openRouterId'] and m['openRouterId'] in or_pricing:
        website = m['rankings'].get('models', {}).get('website')
        if website:
            price = or_pricing[m['openRouterId']]
            print(f"{m['displayName']}: Elo {website['elo']}, "
                  f"${price['prompt']}/1K input tokens")
```

### Find the best model per arena

```javascript
const { data: models } = await fetch(url, { headers }).then(r => r.json());

const bestPerArena = {};
for (const model of models) {
  for (const [arena, categories] of Object.entries(model.rankings)) {
    for (const [category, stats] of Object.entries(categories)) {
      const key = `${arena}/${category}`;
      if (!bestPerArena[key] || stats.elo > bestPerArena[key].elo) {
        bestPerArena[key] = { model: model.displayName, ...stats };
      }
    }
  }
}
```

### Rank models by tournament win rate

```python
# Sort by 1st-place percentage
for m in sorted(models, key=lambda m: (
    m['rankingDistribution']['first'] / max(m['rankingDistribution']['total'], 1)
), reverse=True)[:10]:
    dist = m['rankingDistribution']
    win_pct = dist['first'] / max(dist['total'], 1) * 100
    print(f"{m['displayName']}: {win_pct:.1f}% 1st place ({dist['first']}/{dist['total']} tournaments)")
```
