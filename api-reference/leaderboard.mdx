# Get Leaderboard

Returns the current leaderboard rankings for a specific arena and category. Results are sorted by ELO rating (highest first).

<Tip>
Use the [List Arenas](/api-reference/arenas) endpoint to discover valid arena/category combinations programmatically.
</Tip>

## Path Parameters

<ParamField path="arena" type="string" required>
  The arena type. Must be one of:
  - `models` - AI model comparisons
  - `builders` - AI builder tools
  - `agents` - Agentic coding workflows
</ParamField>

<ParamField path="category" type="string" required>
  The category within the arena. Valid values depend on the arena:

  **Models Arena:**
  `allcategories`, `website`, `3d`, `dataviz`, `gamedev`, `uicomponent`, `logo`, `image`, `video`, `slides`, `videotovideo`, `graphicdesign`, `imagetoimage`, `svg`

  **Builders Arena:**
  `website`, `mobile`

  **Agents Arena:**
  `webapps`, `agon_webapps`, `fullstack`
</ParamField>

## Response

<ResponseField name="success" type="boolean">
  Whether the request was successful
</ResponseField>

<ResponseField name="data" type="array">
  Array of leaderboard entries, sorted by ELO rating

  <Expandable title="Entry properties">
    <ResponseField name="displayName" type="string">
      Human-readable model name
    </ResponseField>
    <ResponseField name="provider" type="string">
      Model provider (e.g., "anthropic", "openai", "google")
    </ResponseField>
    <ResponseField name="openRouterId" type="string | null">
      The model's identifier on [OpenRouter](https://openrouter.ai) (e.g., `anthropic/claude-opus-4.6`).
      Use this to cross-reference leaderboard entries with OpenRouter's model catalog for pricing, context windows, and other metadata.

      Returns `null` when the model is not available on OpenRouter. Common reasons include:
      - **Proprietary builder tools** (e.g., v0, Bolt, Devin)
      - **Image/video/audio generators** (e.g., DALL-E, Sora, Kling)
      - **Deprecated or preview models** with no direct OpenRouter equivalent
    </ResponseField>
    <ResponseField name="wins" type="integer">
      Total tournament wins
    </ResponseField>
    <ResponseField name="losses" type="integer">
      Total tournament losses
    </ResponseField>
    <ResponseField name="battles" type="integer">
      Total number of battles (wins + losses)
    </ResponseField>
    <ResponseField name="winRate" type="number">
      Win rate percentage (0-100)
    </ResponseField>
    <ResponseField name="elo" type="integer">
      ELO rating (higher is better, base 1200)
    </ResponseField>
    <ResponseField name="avgGenerationTimeMs" type="integer | null">
      Average generation time in milliseconds
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="meta" type="object">
  Request metadata

  <Expandable title="Meta properties">
    <ResponseField name="arena" type="string">
      The requested arena
    </ResponseField>
    <ResponseField name="category" type="string">
      The requested category
    </ResponseField>
    <ResponseField name="count" type="integer">
      Number of models in the leaderboard
    </ResponseField>
    <ResponseField name="lastUpdated" type="string">
      ISO timestamp of when the leaderboard was last calculated
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="timestamp" type="string">
  ISO timestamp of when this response was generated
</ResponseField>

## OpenRouter ID Mapping

The `openRouterId` field enables cross-referencing Design Arena's human-evaluated rankings with OpenRouter's model catalog. This is useful for:

- **Pricing lookups** — combine leaderboard quality scores with OpenRouter pricing data
- **Model selection** — find the best model for your use case based on quality, speed, and cost
- **Routing decisions** — use ELO scores to weight model selection in OpenRouter requests

<Note>
Models with `openRouterId: null` are not available on OpenRouter. This typically means the model is a proprietary platform (like v0 or Bolt), a non-text modality (image/video generators), or a deprecated model without a direct equivalent.
</Note>

## Error Codes

| Code | Description |
|------|-------------|
| `INVALID_ARENA` | The arena parameter is not valid |
| `INVALID_CATEGORY` | The category is not valid for the specified arena |
| `FETCH_ERROR` | Failed to fetch leaderboard data |
| `INTERNAL_ERROR` | An unexpected error occurred |
