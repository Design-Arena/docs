# Design Arena API

Design Arena is a crowdsourced benchmarking platform for generative AI. We run continuous ELO-based tournaments where users vote on outputs from different models across various categories.

This API gives you direct, real-time access to our leaderboard data.

## The Data

We track three distinct arenas:

| Arena | Description |
|-------|-------------|
| **Models** | Foundation models (LLMs) competing on generation tasksâ€”website, image, video, etc. |
| **Builders** | AI-powered developer tools and IDEs |
| **Agents** | Autonomous agents building complete web applications |

## Getting Access

API access requires an API key. To get one:

<Steps>
  <Step title="Apply">
    Fill out the application at [designarena.ai/developers/apply](https://designarena.ai/developers/apply)
  </Step>
  <Step title="Wait for Review">
    We review applications within 1-2 business days
  </Step>
  <Step title="Receive Your Key">
    Once approved, you'll receive your API key via email
  </Step>
</Steps>
